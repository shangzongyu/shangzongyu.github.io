## 介绍

平时使用电脑上网时，网络经常会突然间的卡顿，某些应用又在偷偷的联网下载更新，让人颇为头疼。 跟大家推荐 GitHub 上一款强大的网络监控工具：Sniffnet，可帮助我们监控所有网络流量。 界面简洁直观且支持中文，不仅能实时显示网络连接状态和流量统计，还能识别连接的地理位置和域名信息。

GitHub：[http://github.com/GyulyVGC/sniffnet]

主要功能：

- 选择网络适配器并设置过滤条件进行流量监控
- 实时图表显示网络流量强度和整体统计信息
- 识别 6000+ 种网络服务、协议和恶意软件
- 显示远程主机的域名、ASN 和地理位置信息
- 支持 PCAP 文件导入导出和自定义通知
- 提供多种主题样式和最小化监控功能 跨平台兼容，支持 Windows、macOS 和 Linux 系统，下载安装包即可使用。
