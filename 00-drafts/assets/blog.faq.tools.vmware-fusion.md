## You are running this virtual machine with side channel mitigations enbaled

![[../../00 old/assets/vmware-fusion-faq00.png]]

解决方案：<https://kb.vmware.com/s/article/79832>

### **On Fusion**

- Start Fusion
- Virtual Machine should be Shut Down
- Go to Virtual Machine > Settings > Advanced
- Check "Disable Side Channel Mitigations"

![[vmware-fusion-faq01.png]]
