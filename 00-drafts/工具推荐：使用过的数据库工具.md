## 命令行工具

平时管理数据库时，要么使用笨重的图形界面工具，要么在终端里敲复杂的 SQL 命令，操作起来颇为繁琐。如果我们喜欢在终端界面上管理数据库，那么 LazySQL 这个开源项目值得我们一试，它把众多数据库操作功能完美融入到终端界面中。灵感来自知名的 Lazygit 项目，提供了直观的终端界面来管理 MySQL、PostgreSQL、SQLite 等多种数据库。

GitHub：[http://github.com/jorgerojas26/lazysql](https://t.co/EPL5k3VVHK)

主要功能：

- 跨平台支持 macOS、Windows、Linux 系统
- 支持 Vim 键位绑定，操作更加高效
- 多数据库连接管理，可同时管理多个数据库
- 内置 SQL 编辑器，支持语法高亮和快速执行
- 表格数据浏览和编辑，支持行级增删改操作
- 支持 MySQL、PostgreSQL、SQLite、MSSQL 等主流数据库通过 Homebrew 或 Go 包管理器即可快速安装，适合经常需要在终端上工作的开发者使用。
