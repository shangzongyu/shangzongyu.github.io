## Perplexity Ask MCP Server

A Model Context Protocol Server connector for Perplexity API, to enable web search without leaving the MCP ecosystem.

https://mcp.so/server/perplexity/ppl-ai

## Time

A Model Context Protocol server that provides time and timezone conversion capabilities. This server enables LLMs to get current time information and perform timezone conversions using IANA timezone names, with automatic system timezone detection.

https://mcp.so/server/time/modelcontextprotocol

### Awesome MCP Servers

一个由社区维护的精选集合，收录来自不同领域（浏览器控制、数据库、工具集成等）的 MCP 服务器。​

- 网址：https://mcpservers.org

- 特点：
  - 按类别浏览（如 Cloudflare、Browserbase、DeepWiki、Next.js）
  - 每个服务附带描述和使用说明
  - UI 简洁，方便搜索

### PulseMCP Directory

网址：https://www.pulsemcp.com/servers

一个每日更新的索引平台，目前已收录超 6300+ 个 MCP Server。​

特点：

- 支持搜索、筛选与 GitHub 链接跳转
- 提供兼容的 MCP Clients 一览（包括 Claude、Cursor、Windsurf）
- 支持 RSS 更新提醒

### 5. Smithery MCP Hub

- 网址：https://smithery.ai

老牌 MCP 搜索与安装中心，收录超 2200 个 MCP Server 并支持一键复制 install 命令。​

特点：

- 附带 GitHub 链接查看源代码
- 提供可用性标识与命令复制功能
- 常与 PulseMCP 并列使用以覆盖更多资源
