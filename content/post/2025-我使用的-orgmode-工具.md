## 前言

2025 年这个时代，Markdown 文档盛行，大部分的技术文档都使用 Markdown 进行编写，现在的 AI 对 Markdown 的支持也很好。不过有一个文档格式 org 文档，我自己认为它比 Markdown 文档更好用。但是由于它通常被用在 Emacs，导致除 Emacs 用户之外的其他的很少接触这个更好的文档，今天不会介绍他的语法，说下我自己使用 Org 文档的几个工具。

<!--more-->

Org Website : <https://orgmode.org/>

## 工具

工具分为电脑端和手机端。

### 电脑端

#### Emacs

毫无疑问就是 Emacs 自己了，但是 Emacs 的配置比较复杂一些，因此出现了一些社区驱动的 Emacs 发行版，这里主要说两个。

![spacemacs-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_spacemacs-00.png)

[SpaceEmacs](https://www.spacemacs.org/) - 对新手友好

![doom-emacs-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_doom-emacs-00.png)

[DoomEmacs](https://doomemacs.org) - 功能和 SpaceEmacs 类似，但是性能更好一些。

> PS: 我现在主要使用这个。

#### Logseq

![logseq-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_logseq-00.jpeg)

这个工具支持 Markdown 和 Org，我主要使用 Org 模式，这样就可以和我的其他使用 Org 的工具一起使用。

- Website: <https://logseq.com/>

### 手机端

> 由于使用苹果手机，因此主要说苹果手机的软件。

### Plain Org

![plain-org-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_plain-org-00.jpeg)

- AppStore: <https://apps.apple.com/us/app/plain-org/id1578965002>

主要功能：使用 Org 管理任务。

### Flat Habits

![flat-habits-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_flat-habits-00.jpeg)

- AppStore：<https://apps.apple.com/us/app/flat-habits/id1558358855>

主要功能: 习惯记录，只不过文本是 org。

### _scratch_

![scrtatch-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_scrtatch-00.png)

- AppStore：https://apps.apple.com/us/app/scratch/id1671420139

主要功能：主要记录临时的想法。

### Beorg

![beorg-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_beorg-00.jpeg)

- AppStore：<https://apps.apple.com/us/app/beorg-to-do-list-agenda/id1238649962>

主要功能：使用 Org 管理任务，但是功能更丰富。

- 支持 iCloud/OneDrive 等云盘
- 支持同步 Apple Reminder 到一个 Org 文档
- 支持同步 Apple Calender 到一个 Org 文档
- 等等

> PS: 软件本身是免费的，不过一些功能需要购买。

##### Logseq

![logseq-00](https://raw.githubusercontent.com/shangzongyu/blog-image/main/2025/upgit_20251127_logseq-00-4215978.jpeg)

Logseq 也有手机端，但是我自己使用过程中感觉他不好用，它是把电脑端的使用 UI 直接搬到手机端了，这样手机端使用不是很方便，但还是要说一些。

- AppSotre: <https://apps.apple.com/us/app/logseq/id1601013908>

## 最后说明

由于 Org 文档很适合做任务管理，因此为主要用它做任务管理。
