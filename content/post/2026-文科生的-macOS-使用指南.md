## Quick Look 代码预览增强

有时候不想打开代码编辑器，快速预览代码文件，会选择用 macOS 系统自带的 Quick Look 工具查看，但默认只能显示纯文本，阅读体验不佳。

### Syntax Highlight 扩展

安装 Syntax Highlight 这个 Quick Look 扩展，当按下空格键能预览带语法高亮的代码。支持超过 200 种编程语言和文件格式，还能自定义配色方案、字体、行号显示等各种渲染选项。

**GitHub**：[http://github.com/sbarex/SourceCodeSyntaxHighlight](http://github.com/sbarex/SourceCodeSyntaxHighlight)

### 主要功能

- 支持 200+ 种编程语言，包括 Swift、Python、JavaScript、Go 等主流语言
- 集成到 macOS Quick Look，选中文件按空格即可预览高亮代码
- 可自定义明暗主题配色、字体样式、行号显示等渲染选项
- 支持 Git 和 Mercurial 版本控制，能高亮显示代码变更
- 提供命令行工具，支持批量转换代码文件为 HTML 或 RTF 格式
- 可处理无扩展名的纯文本文件，智能识别文件类型

### 安装步骤

1. 下载应用后拖到 Applications 文件夹
2. 首次运行需在系统偏好设置中启用 Quick Look 扩展
3. 重启 Finder
4. 选中任意代码文件，按空格键即可预览

## 文字处理工具

### 写作环境优化

#### 中文输入法

**系统自带输入法**：功能完善，无需额外安装，足够用和够安全。

#### 字体设置

推荐使用以下字体提升阅读和写作体验：

- **中文**：[霞鹜文楷](https://github.com/lxgw/LxgwWenKai)、[思源宋体](https://github.com/adobe-fonts/source-han-serif)、苹方字体
- **英文**：SF Pro、Helvetica Neue

### OCR 文字识别

#### macOS 内置 OCR

macOS 内置了强大的 OCR 功能，可用于图片文字识别：

1. 打开"预览"应用
2. 打开包含文字的图片
3. 点击"工具" > "显示标记工具栏"
4. 点击文本识别图标

#### 第三方 OCR 工具

- **TextSniper**：[https://textsniper.app/](https://textsniper.app/) - 快速截图识别，支持多语言
- **PearOCR**：[https://ocr.space/](https://ocr.space/) - 在线 OCR 服务，识别准确率高
- **白描**：[https://web.baimiaoapp.com/](https://web.baimiaoapp.com/) - 专业的中文 OCR 工具
