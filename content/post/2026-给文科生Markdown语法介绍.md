## 什么是 Markdown？

Markdown 是一种轻量级的标记语言，用简单的符号来格式化文本。

**为什么适合文科生？**
- 学习简单，几分钟就能上手
- 专注内容，不用关心排版
- 支持 LaTeX 数学公式，适合学术写作
- 纯文本文件，可以在任何编辑器中打开

## 推荐编辑器

最适合做笔记的工具。

**特点**：
- 本地存储，数据安全
- 强大的链接功能，适合知识管理
- 丰富的插件生态
- 支持实时预览

### Typora

所见即所得的 Markdown 编辑器。

**特点**：
- 编辑即预览，没有分割窗口
- 界面简洁美观
- 支持导出 PDF、Word 等格式

### VS Code

功能强大的代码编辑器。

**特点**：
- 支持大量插件
- 语法高亮和自动补全
- 适合需要更多自定义的用户

## 基础语法

### 标题

用 `#` 表示标题，数量表示级别。

```markdown
# 一级标题
## 二级标题
### 三级标题
```

效果：

# 一级标题
## 二级标题
### 三级标题

### 段落

段落之间空一行即可。

```markdown
这是第一段。

这是第二段。
```

### 文字格式

- **粗体**：用 `**文字**` 或 `__文字__`
- *斜体*：用 `*文字*` 或 `_文字_`
- ~~删除线~~：用 `~~文字~~`

```markdown
**这是粗体**
*这是斜体*
~~这是删除线~~
```

效果：
**这是粗体**
*这是斜体*
~~这是删除线~~

### 列表

#### 无序列表

用 `-`、`*` 或 `+` 开头。

```markdown
- 第一个项目
- 第二个项目
- 第三个项目
```

效果：
- 第一个项目
- 第二个项目
- 第三个项目

#### 有序列表

用数字加 `.` 开头。

```markdown
1. 第一步
2. 第二步
3. 第三步
```

效果：
1. 第一步
2. 第二步
3. 第三步

#### 嵌套列表

缩进 2 或 4 个空格。

```markdown
1. 第一点
   - 子项目 A
   - 子项目 B
2. 第二点
```

效果：
1. 第一点
   - 子项目 A
   - 子项目 B
2. 第二点

### 链接

#### 文字链接

```markdown
[显示文字](链接地址)
```

例如：

```markdown
[百度](https://www.baidu.com)
```

效果：[百度](https://www.baidu.com)

#### 引用链接

先定义链接，后面使用。

```markdown
[百度]: https://www.baidu.com

这里用 [百度]
```

### 图片

```markdown
![替代文字](图片链接)
```

例如：

```markdown
![示例图片](https://via.placeholder.com/150)
```

效果：
![示例图片](https://via.placeholder.com/150)

### 引用块

用 `>` 表示引用，可以嵌套。

```markdown
> 这是一段引用。
>
> > 这是嵌套引用。
```

效果：

> 这是一段引用。
>
> > 这是嵌套引用。

### 代码

#### 行内代码

用反引号 `` ` `` 包裹。

```markdown
这是一段文字，中间有 `code`。
```

效果：这是一段文字，中间有 `code`。

#### 代码块

用三个反引号 ``` 包裹，可以指定语言。

```markdown
```python
def hello():
    print("Hello, World!")
```
```

效果：

```python
def hello():
    print("Hello, World!")
```

## 高级语法

### 表格

使用 `|` 分隔列，`-` 分隔表头。

```markdown
| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |
```

效果：

| 姓名 | 年龄 | 城市 |
|------|------|------|
| 张三 | 25   | 北京 |
| 李四 | 30   | 上海 |

**对齐方式**：

```markdown
| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 内容   | 内容 | 内容   |
```

效果：

| 左对齐 | 居中 | 右对齐 |
|:-------|:----:|-------:|
| 内容   | 内容 | 内容   |

### 脚注

定义脚注并在文中引用。

```markdown
这里有一句话[^1]。

[^1]: 这是脚注的内容。
```

效果：这里有一句话[^1]。

[^1]: 这是脚注的内容。

### 分隔线

用 `---`、`***` 或 `___`。

```markdown
上面内容

---

下面内容
```

效果：

上面内容

---

下面内容

### 任务列表

用 `- [ ]` 表示未完成，`- [x]` 表示已完成。

```markdown
- [x] 完成的任务
- [ ] 未完成的任务
- [ ] 另一个未完成的任务
```

效果：
- [x] 完成的任务
- [ ] 未完成的任务
- [ ] 另一个未完成的任务

## 数学公式

使用 LaTeX 语法，用 `$` 包裹。

### 行内公式

```markdown
这是行内公式 $E = mc^2$。
```

效果：这是行内公式 $E = mc^2$。

### 公式块

```markdown
$$
\int_{a}^{b} f(x) dx
$$
```

效果：

$$
\int_{a}^{b} f(x) dx
$$

### 常用数学符号

| 符号 | Markdown |
|------|----------|
| 分数 | `\frac{a}{b}` |
| 上标 | `x^{2}` |
| 下标 | `x_{i}` |
| 根求 | `\sqrt{x}` |
| 希腊字母 | `\alpha, \beta, \gamma` |
| 求和 | `\sum_{i=1}^{n}` |
| 积分 | `\int_{a}^{b}` |

## 学术写作

### 引用格式

虽然 Markdown 没有内置的引用功能，但可以用脚注模拟。

```markdown
根据研究[^1]，这种现象很常见。

[^1]: Smith, J. (2020). *The Art of Research*. Academic Press.
```

### 交叉引用

Obsidian 等编辑器支持用 `[[链接]]` 创建笔记间的链接。

```markdown
这是关于 [[另一个笔记]] 的内容。
```

## 实用技巧

### 转义字符

如果需要显示特殊符号，用 `\` 转义。

```markdown
\* 不是斜体
\# 不是标题
```

### HTML 支持

Markdown 支持直接插入 HTML。

```markdown
<details>
<summary>点击展开</summary>

这里是被折叠的内容。
</details>
```

### 注释

用 `<!-- -->` 添加注释，不会显示。

```markdown
<!-- 这是注释，不会显示 -->

这是可见内容。
```

## 练习建议
