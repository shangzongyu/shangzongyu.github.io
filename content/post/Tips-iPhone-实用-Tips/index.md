## 如何在 iOS 上查看您的旅行记录并删除它

该功能被深深隐藏在设置中，目的很可能是为了让用户不会意外地发现它。

1. 打开 “隐私” 部分，然后 “位置服务”，向下滚动到 “系统服务”。
2. 在那里，点击 “重要位置”，并使用 Face ID 进入。
3. 为了限制跟踪，只需将 “重要位置” 的切换开关关闭。
4. 要删除历史记录，点击 “清除历史”。

## 如何从 iPhone 上的视频中提取音频

1️. 安装 [MyMp3](https://apps.apple.com/us/app/mymp3-convert-videos-to-mp3-and-best-music-player/id834818663) 应用程序。
2️. 启动它并点击右上方的放大镜图标。
3️. 点击 "Choose video from gallery"。
4️. 选择您想提取音频的视频。
5️. 点击三个点，选择 "打开"。
6️. 选择 "保存到文件"。

> 提取的音频将是 MP3 格式的。

## iPhone 用户除了使用干净的备用机以外，还可以利用屏幕时间里的 app 分级来隐藏所有非 Apple 原生 app

设置方法：

设置 -> 屏幕使用时间 -> 内容与隐私访问限制 -> 内容访问限制 -> app ->  不允许
Setting -> Screen Time -> Content & Privacy Restrictions -> Content Restrictions -> app -> Don't Allow

返回主页之后大部分 App 都不在了，留下的基本都是系统自带的，使用查找功能也看到其他 app。

相对而言这个办法既不需要删除 app，也不需要额外配置备用机。

只是配置起来稍显麻烦，而且需要时间。

## 快速锁定 iPhone 的方法

连续续快速按5下锁屏关机键，即可快速锁定 iPhone，Face ID 和 Touch ID 均会失效，需要输入 iPhone 的密码才能重新启用面容解锁功能。

在这个界面，还可以滑动关机，也是快速关机 iPhone 的方法。
