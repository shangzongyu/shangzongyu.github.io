自己 2025年常用工具整理。

<!--more-->

- **编辑器**
  - [Zed](https://zed.dev)：快速打开文件和项目的现代代码编辑器。
  - [Antigravity](https://antigravity.google)：主要写代码，集成 AI 的编辑器。
  - [Typora](https://typora.io)：写 Blog 必备的所见即所得 Markdown 编辑器。
  - [Vim](https://www.vim.org)/[Neovim](https://neovim.io)：服务器代码修改、临时修复 Bug。
  - [MiaoYan](https://github.com/tw93/MiaoYan)：本地优先、好看易用的 Markdown 编辑器，但对我来说不太适合作为主力。
  - [Obsidian](https://obsidian.md)：学习笔记和 Journal 记录的知识库工具。

- **浏览器**
  - [Helium](https://helium.computer)：今年新发现的浏览器，主要用于开发环境（开源仓库在 [GitHub](https://github.com/imputnet/helium)）。
  - [Comet](https://www.perplexity.ai/comet)：今年主力的浏览器，AI 原生。
  - [Zen Browser](https://zen-browser.app)：用于替代 Arc 的 Firefox 系浏览器，但最后被 Comet 替代。

- **终端**
  - [Kitty](https://sw.kovidgoyal.net/kitty/)：前半年主要使用 Kitty。
  - [Ghostty](https://ghostty.org)：后半年主要使用的终端（原文 Chostty），体验更好。（GitHub: https://github.com/ghostty-org/ghostty）

- **AI**
  - **工具**
    - [Claude Code](https://claude.ai/code)：写代码、重构和读代码的 AI 助手。
  - **模型**
    - [GLM](https://www.bigmodel.cn)：今年以 GLM 为主，购买了 Coding Plan 套餐，结合 Claude Code 使用。

- **版本控制**
  - **Subversion**：由于工作的变动，今年工作主要使用 SVN。
    - [SnailSVN](https://snailsvn.com)：macOS 的 GUI 工具，支持 Finder，类似 Windows 的[小乌龟](https://tortoisesvn.net/)。
    - [SmartSVN](https://www.smartsvn.com)：年底自己使用的另外一个 SVN GUI 工具，很好用，比 SnailSVN 好用，缺点就是付费。
  - **Git**
    - [Codeberg](https://codeberg.org)：今年发现的新的 Git 仓库管理网站，主要是社区驱动，作为自己 GitLab 之外的备份。
    - [lazygit](https://github.com/jesseduffield/lazygit)：今年主要的 Git 工具，很少使用 Fork/Tower 这样的 GUI 工具。

- **科学上网**
  - [Clash Party](https://clashparty.org)：使用 Clash Party 替代老旧的 ClashX。

- **其他**
  - [LaunchBar](https://www.obdev.at/products/launchbar)：替换了 RayCast，我只想要一个启动器，不想那么复杂的 AI 集成。
  - [Mole](https://github.com/tw93/Mole)：清理 Mac 系统的软件，好用推荐，适合我这种命令行操作的人。
  - [Beyond Compare](https://www.scootersoftware.com/): 今年主要的软件对比工具。
  - [Meld](https://meldmerge.org)：Beyond Compare 的备胎，跨平台开源 diff 工具。
  - [picList](https://piclist.cn/)：自己写 Blog 照片管理工具（自用项目，按需加上自己的仓库地址）。
  - [Apple Reminder](https://apps.apple.com/app/reminders/id1108187841)：生活任务管理。
  - [Apple Notes](https://apps.apple.com/app/notes/id1110145109)：日常记录与简单文档。
  - [Pro Notes](https://www.pronotes.app/)：让 Apple Notes 可以使用类似 Markdown 语法 + AI 的增强应用。
  - [AyuGram](https://ayugram.com)：替代电脑端的 TG 软件。
  - [SwiftGram](https://swiftgram.app/)：替代 iOS 端的 TG 软件（一般通过 TestFlight / App Store 分发）。
  - [Bruno](https://www.usebruno.com)：用于替代 Postman 的 API Client，购买了付费版本。
  - [KeePass](https://keepass.info)：自己的密码管理主力工具（电脑端）。
  - Fantasy Pass：自己的密码管理主力工具（iOS 端，App Store 应用）。
  - Apple Password：管理 Apple 管的密码和账号，以及 Passkey（iOS / macOS 系统内置密码管理）。
