## QuickLook

有时候不想打开代码编辑器，快速预览代码文件，会选择用 macOS 系统自带的 Quick Look 工具查看，但只能显示纯文本挺难阅读的。

可以安装 Syntax Highlight 这个 Quick Look 的扩展，当按下空格键能预览带语法高亮的代码。支持超过 200 种编程语言和文件格式，还能自定义配色方案、字体、行号显示等各种渲染选项，甚至支持 Git 版本控制的差异高亮。

GitHub：<http://github.com/sbarex/SourceCodeSyntaxHighlight>

主要功能：

- 支持 200+ 种编程语言，包括 Swift、Python、JavaScript、Go 等主流语言；
- 集成到 macOS Quick Look，选中文件按空格即可预览高亮代码；
- 可自定义明暗主题配色、字体样式、行号显示等渲染选项；
- 支持 Git 和 Mercurial 版本控制，能高亮显示代码变更；
- 提供命令行工具，支持批量转换代码文件为 HTML 或 RTF 格式；
- 可处理无扩展名的纯文本文件，智能识别文件类型。
  

下载应用后拖到 Applications 文件夹，首次运行需在系统偏好设置中启用 Quick Look 扩展即可使用。
