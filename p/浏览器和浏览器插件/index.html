我的常用浏览器

  • Zen Browser
    基于 Firefox 的改造版本，主打简洁和定制化。适合喜欢自由折腾的用户。
    官网：Zen Browser

  • Arc Browser
    Chrome 内核重新设计，主打多空间、多配置和隐私保护，非常适合多任务场景。
    官网：Arc Browser 特色功能包括 “Spaces”、“Profiles”、“Split View”，以及贴心的隐私保护。[1]

  • Comet (AI 浏览器) Perplexity 推出的新型 AI 浏览器，集成智能助手，具备智能分类、自动整理、快速邮件回复等能力。让你在处理繁杂信息时更加轻松高效。[2]

    • 官网：Comet

  • Helium

    • 官网: Heliu

    • 简洁，无冗余，安全，注重隐私，去除了各种 google 跟踪的东西，关键是没有 AI。资源占用低，不必忍受 Chrome 的内存占用，基于 ungoogled-chromium ，支持所有 Chrome 扩展，也支持导入书签。支持 split view 每次打开一个非常干净的界面就很舒服，我已经能做到大约一大半时间都在使用它了。 极力推荐各位极简爱好者使用一下。
