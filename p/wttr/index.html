# 使用方法
curl wttr.in/你要查询的城市

# 例如查询深圳天气
curl wttr.in/shenzhen

# 想以中文方式显示的话只要改为
curl zh.wttr.in

# 例如查询深圳天气
curl zh.wttr.in/shenzhen

# 更多使用方法可以输入
curl wttr.in/:help
