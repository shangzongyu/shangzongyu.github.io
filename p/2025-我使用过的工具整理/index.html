2025 我使用过的工具整理

Jan 02, 2026
阅读时长: 3 分钟

我的 2025年常用工具整理。

  • 编辑器
    • Zed：快速打开文件和项目的现代代码编辑器。
    • Antigravity：主要写代码，集成 AI 的编辑器。
    • Typora：写 Blog 必备的所见即所得 Markdown 编辑器。
    • Vim/Neovim：服务器代码修改、临时修复 Bug。
    • MiaoYan：本地优先、好看易用的 Markdown 编辑器，但对我来说不太适合作为主力。
    • Obsidian：学习笔记和 Journal 记录的知识库工具。
    • CotEditor：轻量纯文本编辑器。
  • 浏览器
    • Helium：今年新发现的浏览器，主要用于开发环境（开源仓库在 GitHub）。
    • Comet：今年主力的浏览器，AI 原生。
    • Zen Browser：用于替代 Arc 的 Firefox 系浏览器，但最后被 Comet 替代。
  • 终端
    • Kitty：前半年主要使用 Kitty。
    • Ghostty：后半年主要使用的终端（原文 Chostty），体验更好。（GitHub: https://github.com/ghostty-org/ghostty）
  • AI
    • 工具
      • Claude Code：写代码、重构和读代码的 AI 助手。
    • 模型
      • GLM：今年以 GLM 为主，购买了 Coding Plan 套餐，结合 Claude Code 使用。
  • 版本控制
    • Subversion：由于工作的变动，今年工作主要使用 SVN。
      • SnailSVN：macOS 的 GUI 工具，支持 Finder，类似 Windows 的小乌龟。
      • SmartSVN：年底自己使用的另外一个 SVN GUI 工具，很好用，比 SnailSVN 好用，缺点就是付费。
    • Git
      • Codeberg：今年发现的新的 Git 仓库管理网站，主要是社区驱动，作为自己 GitLab 之外的备份。
      • lazygit：今年主要的 Git 工具，很少使用 Fork/Tower 这样的 GUI 工具。
  • 数据库管理
    • Redis GUI 工具
      • Another Redis Desktop
      • Tiny DRM
      • Redis Insight: Redis 官方的 GUI 工具
    • MySQL GUI 工具
      • Navicat Lite: Navicat 的 Lite 版本，提供基础的功能，但是基本够用。
      • DBEaver：社区驱动的数据库管理工具。
      • Table Plus: 自认为最漂亮 MySQL GUI。
    • PostgreSQL GUI
      • pgAdmin 4
      • Postico 2
  • 科学上网
    • Clash Party：使用 Clash Party 替代老旧的 ClashX，主要在 mac 上使用。
    • Shadowrocket：iOS 代理神器，最良心的代理软件。
  • 其他
    • LaunchBar：替换了 RayCast，我只想要一个启动器，不想那么复杂的 AI 集成。
    • Mole：清理 Mac 系统的软件，好用推荐，适合我这种命令行操作的人。
    • PrettyClean：清理垃圾释放空间，可以和 Mole 搭配使用，一个命令行，一个 GUI。
    • Beyond Compare: 今年主要的软件对比工具。
    • Meld：Beyond Compare 的备胎，跨平台开源 diff 工具。
    • picList：自己写 Blog 照片管理工具（自用项目，按需加上自己的仓库地址）。
    • Apple Reminder：生活任务管理。
    • Apple Notes：日常记录与简单文档。
    • Pro Notes：让 Apple Notes 可以使用类似 Markdown 语法 + AI 的增强应用。
    • AyuGram：替代电脑端的 TG 软件。
    • SwiftGram：替代 iOS 端的 TG 软件（一般通过 TestFlight / App Store 分发）。
    • Bruno：用于替代 Postman 的 API Client，购买了付费版本。
    • KeePass：自己的密码管理主力工具（电脑端）。
    • Fantasy Pass：自己的密码管理主力工具（iOS 端，App Store 应用）。
    • Apple Password：管理 Apple 管的密码和账号，以及 Passkey（iOS / macOS 系统内置密码管理）。
    • Cherry Studio：跨平台多模型桌面客户端，今年前半年还是很喜欢的，后半年不知道啥原因，在 Mac 系统系统的卡的不行，慢慢就不用了，我喜欢这个的主要原因就是可以在一个 App 里面使用多个模型。
    • Loop：开源免费的分屏管理窗口工具，很好用。
    • CC Switch：一键切换Claude Code供应商配置。
    • MenubarX：菜单栏里打开网页小窗，刚出来我就关注了，用的比较少。
    • Ice：整理/隐藏菜单栏图标，在 mac 升级到最新的 26 时候，有不少 BUG，等待修改，不过开源免费值得使用。
    • Kando：圆盘式快捷启动器，占用资源有些大。
    • OrbStack：更快的 Docker/容器环境，对开发者友好。
